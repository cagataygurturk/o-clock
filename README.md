# o-clock [![NPM version](https://badge.fury.io/js/o-clock.svg)](http://badge.fury.io/js/o-clock)

> Simple utility for displaying the time in 12-hour clock format.

## Install
## Install with [npm](npmjs.org)

```bash
npm i o-clock --save
```

## Run tests

```bash
npm test
```

## Usage

```js
var oclock = require('o-clock');

oclock(new Date());
//=> 12:59 PM
```
